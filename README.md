# Описание приложения Bin

### Русский
Это сайт-мессенджер для обмена сообщениями, фотографиями и видео между пользователями, который включает следующие функции:

- Чаты и история чатов
- Профили пользователей с возможностью просмотра своего профиля и профилей других пользователей
- Кошелёк и конвертация валют
- Поддержка двух языков: украинского и английского
- Подписки:
  - **Bin+** — 4,99 $ или 199,99 ₴
  - **Bin Premium** — 9,99 $ или 399,99 ₴
- Возможность смены темы:
  - бесплатная версия → белая и тёмная
  - с подпиской (начиная с Bin+) → + розовая и зелёная
- Раздел безопасности, где можно изменить пароль (нужно знать текущий пароль)

### Українською
Це месенджер-сайт для обміну повідомленнями, фотографіями та відео між користувачами, який має такі функції:

- Чати та історія чатів
- Профілі користувачів і перегляд власного профілю та профілів інших користувачів
- Гаманець і конвертація валют
- Підтримка двох мов: української та англійської
- Підписки:
  - **Bin+** — 4,99 $ або 199,99 ₴
  - **Bin Premium** — 9,99 $ або 399,99 ₴
- Можливість зміни теми:
  - безкоштовна версія → біла та чорна
  - з підпискою (починаючи з Bin+) → + рожева та зелена
- Розділ безпеки, де можна змінити пароль (знаючи поточний пароль)

### English
This is a messenger website for exchanging messages, photos, and videos between users, featuring the following functions:

- Chats and chat history
- User profiles with the ability to view your own profile and other users’ profiles
- Wallet and currency conversion
- Support for two languages: Ukrainian and English
- Subscriptions:
  - **Bin+** — $4.99 or 199.99 UAH
  - **Bin Premium** — $9.99 or 399.99 UAH
- Theme customization:
  - free version → white and dark
  - with subscription (starting from Bin+) → + pink and green
- A security section where users can change their password (must know the current password)
